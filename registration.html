Registration

CC 2019 is an event co-located with CGO 2019 and the registration happens via the CGO 2019 registration system.

For attending CC 2019 a registration for the 2-Days Workshop/Tutorial option is required.

For attending CC 2019 and CGO 2019 a registration for the 2-Days Workshops/Tutorial option and the CGO Main Conference is required.

Early registration ends on any time on earth.
